# Rodando o Projeto (_frontend_)

Instale o npm v6.4+ e, na pasta `./frontend`, execute

```shell-session
npm install
```

e, após as dependências instaladas, rode

```shell-session
npm run serve
```

PRONTO! Agora é só acessar `localhost:8080` (se tudo está rodando com as opções padrão)
